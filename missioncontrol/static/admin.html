      Test buy
      
                        
      Smoke-test the checkout flow against Good360 with our master account and a card you supply. Used to confirm the buy pipeline is healthy without burning real customer funds.
      
                    
      
                
      

                
      
                    
                    
      
                        DevTools MCP agent — drives a real Chrome session against Good360 via OpenAI's Agents SDK.
                        
                            Login uses the master scan account (SCAN_GOOD360_*). The buyer name + email below are typed into Good360's checkout fields. The card you supply is the test card the agent will fill on the payment page. By default Live submit is off — the agent stops before clicking Place Order and returns DRY_RUN. To capture a real "card declined" error, tick Live submit (requires DEVTOOLS_AGENT_ALLOW_LIVE_PURCHASE + DEVTOOLS_AGENT_ALLOW_SECRETS_TO_MODEL = true in Settings). Only one test runs at a time. Tokens cost ~$0.05–$0.50 per run.
